# HyperReal Studio

Plataforma SaaS de generación de contenido IA hiperrealista.

## Deploy en Vercel

1. Sube este repo a GitHub
2. Conecta el repo en vercel.com
3. Deploy automático

## Stack
- React + Vite
- Supabase (Auth + DB)
- fal.ai (Nano Banana 2 + Kling 3.0) — Fase 3
- Stripe (pagos) — Fase 3
